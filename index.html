Look up any biological identifier

Paste any biological identifier — UniProt accession, Ensembl ID, RefSeq, HGNC, GO term, Reactome pathway, ChEMBL, rsID, gene symbol, and more. Bio.Sync detects the type automatically and returns all cross-database mappings, aliases, species, and provenance. Input is case-insensitive. Paste anywhere on the page to auto-fill.

Design note — Bio.Sync uses the visual language of mid-century Swiss Federal Railways printed timetables: white paper, typographic grid, colour only to encode meaning, no ornament. A timetable tells you all connections from a departure point and which are direct; Bio.Sync does the same for biological identifiers. Primary names are set in Fraunces, a variable optical-size serif that reads with authority at display sizes; identifiers in JetBrains Mono.
