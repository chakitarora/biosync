Design note — The visual language of Bio.Sync is borrowed from mid-century Swiss Federal Railways printed timetables: white paper, typographic precision, colour used only to encode meaning, no ornament. A timetable tells you all the connections from your departure point and which are direct; Bio.Sync does the same for biological identifiers. Primary type is set in Fraunces (a variable optical-size serif); identifiers in JetBrains Mono.
